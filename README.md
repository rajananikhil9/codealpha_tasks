# CodeAlpha Internship Projects

This repository contains the projects completed as part of the **CodeAlpha Internship Program**.

## Projects

### 1. Cloud-Based Bus Pass System

A cloud-based bus pass management system that allows users to apply for bus passes online while providing administrators with tools to manage applications, verify passes, and monitor records.

**Technologies Used**

* Python Flask
* HTML5
* CSS3
* JavaScript
* AWS DynamoDB
* AWS Lambda
* JWT Authentication

**Features**

* Online Bus Pass Application
* QR Code Verification
* Pass Tracking
* Admin Dashboard
* Route Management
* Reports & Analytics
* Cloud Database Integration

---

### 2. Cloud-Based Data Redundancy Removal System

A cloud-based application that detects duplicate and false-positive records before storing data, ensuring data consistency and reducing redundancy.

**Technologies Used**

* Python Flask
* HTML5
* CSS3
* JavaScript
* AWS DynamoDB
* JWT Authentication

**Features**

* Admin Login
* Add Records
* Duplicate Detection
* False Positive Detection
* Dashboard Statistics
* Record Search
* AWS Cloud Database
